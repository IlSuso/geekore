# \# Geekore

# 

# \## Stack

# 

# \- \*\*Framework\*\*: Next.js 16 (App Router)

# \- \*\*Styling\*\*: Tailwind CSS

# \- \*\*Database + Auth\*\*: Supabase

# \- \*\*Deploy\*\*: Vercel

# \- \*\*API\*\*: AniList, Steam, IGDB, BoardGameGeek
